# واجهة "أهلاً وسهلاً"

واجهة ترحيبية بسيطة مبنية باستخدام Next.js 14 مع دعم كامل للغة العربية واتجاه الكتابة من اليمين إلى اليسار. الصفحة تعرض رسالة ترحيب، ثلاث مزايا أساسية، وقسمًا ختاميًا يوضح إمكانية التخصيص.

## المتطلبات

- Node.js 18 أو أعلى
- npm

## بدء التشغيل محليًا

```bash
npm install
npm run dev
```

سيتم تشغيل الخادم المحلي على العنوان `http://localhost:3000`.

## البناء للإنتاج

```bash
npm run build
npm start
```

## هيكل المشروع

```
├── app/
│   ├── globals.css      # الأنماط العامة
│   ├── layout.tsx       # تخطيط الجذر
│   └── page.tsx         # الصفحة الرئيسية الترحيبية
├── public/
│   └── favicon.svg      # الأيقونة
├── package.json
└── tsconfig.json
```

## تخصيص

- عدّل النصوص العربية في `app/page.tsx`.
- حدّث لوحة الألوان أو الخطوط عبر `app/globals.css`.
- أضف أقسامًا أو مكوّنات جديدة ضمن مجلّد `app`.
